## Sobre
O projeto contém tópicos sobre os fundamentos da programação orientada a objetos na linguagem Kotlin.

## Conteúdo do curso
* Classes
* Atributos
* Métodos
* Construtores
* Sobrecarga de construtores
* Sobrecarga de métodos
* Encapsulamento
* Métodos estáticos
* Herança
* Lista de objetos
* Interfaces
